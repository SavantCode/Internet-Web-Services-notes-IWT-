# ‚≠ê **HTML Basic Concepts**

HTML (HyperText Markup Language) is the **foundation of web pages**. These basic concepts are essential to understand how web pages are structured and work.

---

## **1. HTML Elements**

* An **element** is a building block of HTML.
* It consists of a **start tag, content, and end tag**.

**Example:**

```html
<p>This is a paragraph.</p>
```

* `<p>` ‚Üí start tag
* `This is a paragraph.` ‚Üí content
* `</p>` ‚Üí end tag

---

## **2. HTML Tags**

* **Tags** are keywords wrapped in `< >`.
* Tags tell the browser how to display content.
* Most tags come in **pairs**: `<tag>...</tag>`
* Some tags are **self-closing**: `<br>`, `<hr>`, `<img>`

---

## **3. HTML Attributes**

* **Attributes** provide extra information about an element.
* Written inside the **start tag**.

**Example:**

```html
<img src="photo.jpg" width="300" height="200">
```

* `src`, `width`, `height` are attributes of the `<img>` tag.

---

## **4. HTML Document Structure**

A basic HTML page has **three main sections**:

```html
<!DOCTYPE html>
<html>
  <head>
    <title>My Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
    <p>This is a paragraph.</p>
  </body>
</html>
```

### Explanation:

* `<!DOCTYPE html>` ‚Üí defines HTML5
* `<html>` ‚Üí root element
* `<head>` ‚Üí contains metadata, title, scripts
* `<title>` ‚Üí title of the page (shown in browser tab)
* `<body>` ‚Üí visible content

---

## **5. Block vs Inline Elements**

* **Block elements**: start on a new line and take full width

  * Examples: `<div>`, `<p>`, `<h1>`
* **Inline elements**: appear within a line and take only as much width as needed

  * Examples: `<span>`, `<a>`, `<b>`, `<i>`

---

## **6. HTML Comments**

* Used to add notes in the code.
* **Not displayed** in the browser.

```html
<!-- This is a comment -->
```

---

## **7. HTML Entities**

* Special characters that cannot be typed directly.
* Examples:

  * `&lt;` ‚Üí `<`
  * `&gt;` ‚Üí `>`
  * `&amp;` ‚Üí `&`

---

## **8. Semantic HTML (Basic)**

* Tags that **describe meaning** rather than appearance.
* Examples:

  * `<header>` ‚Üí page header
  * `<footer>` ‚Üí page footer
  * `<article>` ‚Üí independent content
  * `<section>` ‚Üí section of content

---

## **9. Advantages of Understanding HTML Basics**

* Helps create well-structured, readable web pages.
* Makes it easier to learn **CSS, JavaScript, DHTML**.
* Essential for web development and exams.

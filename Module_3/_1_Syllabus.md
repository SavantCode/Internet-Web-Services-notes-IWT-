# **Module III: Technologies for Web Applications**

## **1. HTML and DHTML**

* **HTML Basic Concepts**

  * What is HTML
  * Role of HTML in web applications
  * Difference between HTML and DHTML
* **Static vs Dynamic HTML**

  * Static HTML: Fixed content
  * Dynamic HTML: Content changes using scripting (e.g., JavaScript)
* **Structure of HTML Documents**

  * `<html>`, `<head>`, `<body>` tags
  * Document type declaration (DOCTYPE)
* **HTML Elements**

  * Paragraphs, headings, lists, div, span
  * Semantic tags (`<header>`, `<footer>`, `<article>`, `<section>`)
* **Linking in HTML**

  * Anchor tags `<a>`
  * Internal and external links
  * Relative and absolute URLs
* **Anchor Attributes**

  * `href`, `target`, `name`, `title`
* **Image Maps**

  * Concept of clickable areas in images
  * `<map>` and `<area>` tags
* **Meta Information**

  * `<meta>` tags
  * Character encoding, viewport, keywords, description
* **Image Preliminaries**

  * `<img>` tag, attributes: `src`, `alt`, `height`, `width`
* **Layouts**

  * `<div>` and `<span>` for layout
  * Using CSS for layout
* **Backgrounds, Colors, and Text**

  * Background images, background color
  * Text color and alignment
* **Fonts**

  * `<font>` tag (deprecated) and CSS font styles
* **Tables**

  * `<table>`, `<tr>`, `<td>`, `<th>`
  * Rowspan, colspan, table captions
* **Frames and Layers**

  * `<iframe>` for embedding content
  * Layers using CSS `position` and `z-index`
